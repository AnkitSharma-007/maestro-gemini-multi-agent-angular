How it works

Three things make Maestro different

hub

Agents, not a single LLM

Five purpose-built agents collaborate via structured JSON. Each is small, focused, and independently retryable.

auto_awesome

Generative UI, not chat

Outputs render as live Angular widgets: a budget card, a multi-day schedule, a venue panel. You edit the dashboard, not a transcript.

shield_lock

Zero backend

Pure Angular zoneless SPA. Your Gemini API key, prompt, and results never leave the browser tab.

Under the hood

Built on a tight, modern stack

bolt
Angular zoneless

Signals everywhere. Signal-based change detection. No zone.js runtime.

auto_awesome
Gemini structured outputs

Schema-typed JSON streamed in via @google/genai. Lazy-loaded on demand.

verified
Strict TypeScript

Discriminated unions for widget configs. Type-safe agent contracts.

cloud_off
Client-only, CSP-locked

No backend. Strict CSP. AbortController on every stream.
